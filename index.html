About this library

These lesson resources are produced using BELLA — British English Language Learning Assistants, a suite of AI-powered tools built in Google Gemini and Claude. Each lesson includes a self-contained HTML lesson page with embedded audio and an interactive pronunciation coach, alongside printable PDF vocabulary reference sheets.

Lesson pages are designed to be opened directly in any browser. PDF sheets are formatted for A4 printing. Google Doc versions of each lesson are available for upload to Google Notebook LM.
